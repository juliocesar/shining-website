Shining

A toolkit for creating in-browser presentations.

Quickstart

Install via:

$ sudo gem install shining

Create a new presentation:

$ shine mypresentationdir

Intro

Shining helps you creating presentations that you present from a web browser. The presentations that it generates are entirely based in HTML, CSS, and Javascript, with a little help from CSS transforms and transitions, and jQuery.

What makes a presentation?

A presentation is composed of:

  • A configuration file named config.json.
  • Slides, which are plain HTML files in the slides folder.
  • An index file named index.html in the root presentation folder, which is where you start a presentation.

Configuration

Presentation-wide configuration happens in the config.json file. Valid parameters:

  • slides: an array containing the name of the presentation's slides. If you're using the generator to add slides, don't worry about editing it.
  • transitions: a string corresponding to one of the available transition effects, or false disables them altogether.

Effects

Effects are applied on elements by adding CSS classes to them.
